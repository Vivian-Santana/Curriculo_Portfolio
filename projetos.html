Vida Plus

Vida Plus é um sistema de agendamento de consultas médicas, com as funcionalidades de cadastro de paciente, login, reset de senha, agendamento e cancelamento de consulta, listagem de consultas agendadas e listagem de médicos disponíveis. Integrado a API Voll Med desenvolvida com java, Spring Boot e MySQL.

Tecnologias usadas

  • Html
  • CSS
  • Angular
  • TypeScript
  • Node
  • Bootstrap
